# Open HRMS Loan Management


        Helps you to manage Loan Requests of your company's staff.
        

## Installation

```bash
pip install odoo-bringout-cybrosys-ohrms_loan
```

## Dependencies

This addon depends on:
- base
- hr
- payroll
